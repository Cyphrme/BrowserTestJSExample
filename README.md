# ExampleBrowserTestJS
Example of how to use the BrowserTestJS module in a project.

The live example demonstrates what the BrowserTestJS testing page will look like
after following the README/instructions for how to run the unit tests.

This example demonstrates the following:

`example.js` should be viewed as the source code for a given project. This is
the file that contains the function(s) to be unit tested.

`test_unit.js` is the unit test file for a project to interact with the 
`BrowserTestJS` package. This file will import the `example.js` file and run
unit tests on the function(s).

`BrowserTestJS` is added as a submodule in this example repo to demonstrate how
a project is recommended to implement the package.

For an example of how to embed the `BrowserTestJS` testing page directly in
another page, see `iframe.html.example`.

## The source code for `BrowserTestJS` [can be found here.](https://github.com/Cyphrme/BrowserTestJS)
